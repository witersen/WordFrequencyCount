# 英文试卷词频统计软件

### 说明

- C#配合Spire组件开发的英文试卷词频统计和渲染桌面程序 
- 可以帮助统计英文试卷的各单词出现频率生成词库 之后根据词库渲染新试卷
- 本程序使用免费版本的Spire.pdf Spire.xls Spire.doc组件
- 如要重新编译 请自行准备Spire组件
